# hotplace

2주차 맛집 추천 웹 서비스 API입니다.

## sequelize

- reference
  https://sequelize.org/docs/v6/getting-started/

## 초안입니다

```
// 패키지 다운
npm install
// nodemon 실행
npm run dev
// ts-node로 실행
npm start
```

## api_key
- [경기데이터드림}(https://data.gg.go.kr/portal/mainPage.do) 접속 후 회원가입을 진행
- 마이페이지에서 인증키 발급을 통해 apikey을 얻을 수 있습니다.
